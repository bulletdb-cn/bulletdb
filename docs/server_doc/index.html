BulletDB Server

BulletDB works on Windows, Linux, Mac OS X, and has plan for more OS such as Android, IOS, POSIX-compliant OS.

How to start bulletDB server

  1. Download binary package from home page and uncompress.

  2. Run btdb_server on Linux/MacOS or btdb_server.exe on Windows.

./btdb_server
BulletDB 1.0.0.1133
listening on 0.0.0.0:33060

Configure parameters

BulletDB server process uses a configure file which has these parameters:

bind_address:            server socket bind address, default 0.0.0.0

bind_port:               server listen port, default 33060

data_dir:                directory used to save database data

read_cache_size:         read cache size for each database, in bytes, default 128M

max_transaction_memory:  memory limit of each write transaction, in bytes, default 4M

require_lock_timeout:    require lock timeout time, in seconds, default 1 seconds

enable_compress:         enable data compress or not, 1 means yes, default 0

pool_thread_num:         max threads number in thread pool, default 128
