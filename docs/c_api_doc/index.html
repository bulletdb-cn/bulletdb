C API

API Documents for C/C++

bulletdb_connect

Description

The bulletdb_connect function creates a database connection.

Syntax:

HBTDBCONN bulletdb_connect(const char* host, int port);

Parameter:

host: Database server address.

port: Database server listening port.

Return value:

bulletdb_connect returns a handle of connection.

bulletdb_login

Description

The bulletdb_login function login to database server.

Syntax:

int bulletdb_login(HBTDBCONN hconn, const char* user, const char* passwd, const char* database);

Parameter:

hconn: Connection handle created by bulletdb_connect.

user: Account, reserved for later use, any string applies.

passwd: Password of account, reserved for later use, any string applies.

database: Database name should be opened, "" means don't open any database.

Return value:

If no error occurs, bulletdb_login returns 0, otherwise it returns a error code.

bulletdb_execute_command

Description

The bulletdb_execute_command function runs a BulletDB command.

Syntax:

int bulletdb_execute_command(HBTDBCONN hconn, const char* command, const char** json_result, int* json_size);

Parameter:

hconn: Connection handle

command: A pointer to the command string.

json_result: A pointer to receive the command json response string pointer.

json_size: A pointer to receive the json string size.

Return value:

If no error occurs, bulletdb_execute_command returns 0, otherwise it returns a error code.

bulletdb_execute_sql

Description

The bulletdb_execute_sql function runs a SQL statement.

Syntax:

int bulletdb_execute_sql(HBTDBCONN hconn, const char* sql, const char** json_result, int* json_size);

Parameter:

hconn: Connection handle

command: A pointer to the command string.

json_result: A pointer to receive the SQL json response string pointer.

json_size: A pointer to receive the json string size.

Return value:

If no error occurs, bulletdb_login returns 0, otherwise it returns a error code.

bulletdb_error_message

Description

The bulletdb_error_message function gets last error description.

Syntax:

const char* bulletdb_error_message(HBTDBCONN hconn);

Parameter:

hconn: Connection handle

Return value:

This function returns the message string pointer.

bulletdb_get_columns_info

Description

The bulletdb_get_columns_info function get column's name and type of last SQL query if it is a select statement.

Syntax:

int bulletdb_get_columns_info(HBTDBCONN hconn, const column_info** column_info_ptr, int* column_num);

Parameter:

hconn: Connection handle

column_info: A pointer to receive the column_info pointer.

column_num: A pointer to receive the column number.

Return value:

If no error occurs, bulletdb_login returns 0, otherwise it returns a error code.

bulletdb_fetch_records

Description

The bulletdb_fetch_records function fetch records returned by last SQL query, records orgnized by column.

Syntax:

int bulletdb_fetch_records(HBTDBCONN hconn, const column_data** column_data_ptr, int* column_num);

Parameter:

hconn: Connection handle

column_data_ptr: A pointer to receive the column_data pointer.

column_num: A pointer to receive the column number.

Return value:

If no error occurs, bulletdb_fetch_records returns records number, otherwise it returns a negtive error code. It should be called repeatedly until 0 or a negtive value returned.

bulletdb_close

Description

The bulletdb_close function close connection.

Syntax:

void bulletdb_close(HBTDBCONN hconn);

Parameter:

hconn: Connection handle

Return value:

This function has not return value.

bulletdb_number_to_str

Description

The bulletdb_number_to_str function convert a BulletDB number to string.

Syntax:

int bulletdb_number_to_str(const char* number, char* out_buff, int buff_size);

Parameter:

number: Pointer to a BulletDB number.

out_buff: Buffer to received the result string, the buffer should be not less than 32 bytes.

databuff_size: Size of out_buff, in bytes.

Return value:

This function returns bytes copied into out_buff.
