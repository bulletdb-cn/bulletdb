Shell Client

Start shell client

Run btdb_shell on Linux/MacOS or btdb_shell.exe on Windows.

./btdb_shell
BulletDB version 1.0.0.1133
Enter "help;" for help
Enter SQL statements terminated with a ";"
btdb>

Shell options

Usage: ./btdb_shell [OPTIONS] [DATABASE]
Example: ./btdb_shell -h127.0.0.1 -P33060 -utest -ptest db1
OPTIONS include:
   -h             Connect to host, default:127.0.0.1
   -p             Password to use when connecting to server
   -P             Port number to use for connection, default:33060
   -u             User for login
   -help          Show option help

Shell commands

show databases;                           list names of all databases

create database <db_name>;                create database

use <db_name>;                            switch database

desc <table|index|trigger name>;          show schema of table|index|trigger

show tables;                              list names of tables

show index [from table_name];             list index

show triggers [from table_name];          list trigger

import <file_name> <table_name> [sep];    import data from file into table

help;                                     show help message

quit;|exit;                               exit program
