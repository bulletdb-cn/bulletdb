Python API

To use BulletDB python, a bulletdb object should be created first.

import bulletdb
conn = bulletdb.bulletdb()

bulletdb.connect

Description

This method connect database server.

Syntax:

conn.connect(host="127.0.0.1", port=33060, user="", passwd="", database = ""):

Parameter:

host: database server address.

port: database server listening port.

user: account

passwd: password

database: database name

Return value:

If an error occurs, an exception is thrown.

bulletdb.execute_command

Description

This method runs a BulletDB command.

Syntax:

conn.execute_command(cmd)

Example:

conn.execute_command('create database db1')

Return value:

If no error occurs, this method returns a python dict which contains keys: "ret_code", "json_data", "msg", otherwise an exception is thrown.

bulletdb.execute_sql

Description

This method runs a SQL statement.

Syntax:

conn.execute_sql(sql)

Parameter:

sql: SQL string

Example:

conn.execute_sql("select * from stock_history;")
conn.execute_sql("BEGIN TRANSACTION;"
    "UPDATE accounts SET balance = balance - 10 WHERE account_no = '100001';"
    "UPDATE accounts SET balance = balance + 10 WHERE account_no = '200007';"
    "COMMIT;")

Return value:

This method returns a python dict which contains keys: "ret_code", "json_data", "msg".

bulletdb.get_columns_info

Description

This method get column's name and type of last SQL query if it is a select statement.

Syntax:

conn.get_columns_info()

Return value:

If no error occurs, this method returns a python dict which contains keys: "ret_code", "json_data", "msg", otherwise an exception is thrown.

bulletdb.fetch_records

Description

This method fetch records returned by last SQL query, records orgnized by column. This method should be called repeatedly until "records_num" equals 0.

Syntax:

conn.fetch_records()

Return value:

This method returns a python dict which contains keys: "ret_code", "records_num", "columns", "msg".

bulletdb.fetch_json_records

Description

This method fetch records returned by last SQL query, in json format. This method should be called repeatedly until "records_num" equals 0.

Syntax:

conn.fetch_json_records()

Return value:

This method returns a python dict which contains keys: "ret_code", "records_num", "json_data", "msg".

bulletdb.close

Description

This method close database connection.

Syntax:

conn.close()

bulletdb.number_to_str

Description

Convert a BulletDB number to string.

Syntax:

conn.number_to_str(number_bytes);

Parameter:

number_bytes: BulletDB number, python bytes object

Return value:

This function returns result string.
