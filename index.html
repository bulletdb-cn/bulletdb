What is BulletDB?

BulletDB is a fast relational database system. It's mission is to build an extremely fast, real-time, lightweight RDBMS.
Features: SQL, ACID, MVCC, Transaction, Row Lock, Data Compression, Fast Range Query, Server/In-Process Mode, Flat Learning Curve.
